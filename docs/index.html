Mac MD for Windows

Quick Start

  1. Download & extract

    Download the zip from the button above. Right-click → Extract All. No installer, no admin rights needed.

  2. Run MacMD.Win.exe

    Double-click MacMD.Win.exe in the extracted folder. Windows SmartScreen may warn you since the app isn't signed yet — click "More info" → "Run anyway".

  3. Create a project

    Click + New Project in the left column, then + New in the middle column to create a document.

  4. Write Markdown

    Type in the editor (left half of the right column). The preview updates automatically after you pause typing. Try headings, bold, lists, code blocks, links.

  5. Your work auto-saves

    Documents save to a local SQLite database 2 seconds after you stop typing. Close and reopen — your content persists.

Where's My Data?

Your documents are stored locally at:

%LOCALAPPDATA%\MacMD\macmd.db

That's typically C:\Users\YourName\AppData\Local\MacMD\macmd.db. It's a standard SQLite file — you can open it with any SQLite browser if you want to inspect your data.

Known Limitations

  • No keyboard shortcuts yet (Ctrl+N, Ctrl+S, etc. coming in M6)
  • No settings UI — theme palette is defined but selector isn't wired up yet
  • No HTML or PDF export yet (M5)
  • Preview uses a fixed dark theme regardless of the app theme toggle
  • No .md file association — you can't double-click a Markdown file to open it
  • Windows SmartScreen will warn because the app isn't code-signed
  • x64 only for this release (ARM64 coming later)
  • No cloud sync (intentionally — local-first by design)
